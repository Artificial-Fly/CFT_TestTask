# Инструкция по запуску проекта
## Требования к окружению
- **Версия Java:** Java 17
- **Система сборки:** Apache Maven
- **Рекомендуемая версия Maven:** 3.9.11
## Используемые библиотеки и инструменты
В проекте не используются сторонние библиотеки.
Используется:
- стандартная библиотека Java (JDK 17);
- плагины Maven, необходимые для сборки проекта.
  - **maven-jar-plugin 3.2.0** для формирования исполняемого JAR-файла с указанием основного класса
## Сборка проекта
Для сборки проекта выполните команду в корневой директории:
```
mvn clean package
```
После успешной сборки готовый JAR-файл располагается в директории **/target**

## Запуск приложения
Проект является консольным Java-приложением и запускается с командной строки.
- для запуска можно использовать собранный jar-файл в папке **/build**, либо собранный в шаге Сборка проекта

Пример запуска:
```
java -jar util.jar -s -a -p sample- in1.txt in2.txt
```
## Особенности реализации
Проект реализован в виде консольного приложения.
- Входные данные передаются через аргументы командной строки.
- Результаты работы выводятся в стандартный поток вывода (System.out).
- Структура проекта соответствует стандартной Maven-структуре.
